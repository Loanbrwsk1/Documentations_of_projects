# Python CMD Linux

## Descirption

This project aims to create a minimal linux environment in Python.
The basic commands are implemented:

- ls
- cd
- touch
- rm
- mkdir
- rmdir
- exit
- help
- nano
- cat
- pwd
- clear

With a minimal tree:

- /
- /home
- /home/user
- /home/user/Documents
- /home/user/text.txt

## How it works

Write down your commandes in the top bar and press _Return_ to see the result.
